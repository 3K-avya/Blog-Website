# Blog Website

A simple blog website built with React where users can create, read, search, and delete blog posts.

## Features
- Create new blog posts
- View all blog posts
- Read individual blog pages
- Search posts by title
- Delete blog posts
- Data stored using LocalStorage

## Tech Stack
- HTML
- CSS
- JavaScript
- React
- React Router DOM

## Learning Outcomes
- React Components
- React Hooks (useState, useEffect)
- React Router
- LocalStorage
- Basic state management
- Responsive UI design
